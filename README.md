# BankMore APIs

Projeto desenvolvido como desafio técnico.

## Estrutura
- BankMore.ContaCorrente.API
- BankMore.Transferencia.API
- SQLite + Dapper
- JWT Authentication

## Como rodar
1. Clonar o repositório
2. Abrir a solution no Visual Studio
3. Executar as APIs
4. O banco SQLite é criado automaticamente

## Observações
- SQLite utilizado para facilitar execução local
- Estrutura preparada para Docker
